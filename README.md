# 🛡️ SeguraSite Pro Scanner (v0.7)

Scanner de vulnerabilidades web gratuito e open-source, inspirado no Acunetix, focado em proteger sites brasileiros. Feito de forma independente para a comunidade!

## 🚀 Funcionalidades Principais

* **Interface Gráfica (GUI) Intuitiva:** Escaneie sites com poucos cliques, sem precisar decorar dezenas de comandos no terminal.
* **Detecção de SQL Injection (SQLi):** Identifica possíveis brechas em formulários e parâmetros de URL.
* **Detecção de Cross-Site Scripting (XSS):** Verifica se o alvo está vulnerável à injeção de scripts maliciosos.
* **Modos de Escaneamento Flexíveis:** Escolha entre Escaneamento Rápido (1 página) ou Completo (até 10 páginas).
* **Geração de Relatórios:** Exporta os resultados da análise diretamente em formatos profissionais (`.pdf` e `.html`) para facilitar a leitura e o envio para clientes.

## 🛠️ Como Baixar e Instalar

Para rodar o SeguraSite no seu computador, você precisa ter o **[Python](https://www.python.org/downloads/)** instalado.

### Opção 1: Via Download (Recomendado para iniciantes)
1. Role a página para cima e clique no botão verde **`<> Code`**.
2. Selecione a opção **`Download ZIP`**.
3. Extraia a pasta baixada no seu computador.
4. Abra o terminal (CMD) dentro da pasta extraída e execute o arquivo principal:
   ```bash
   python segurasite.py
